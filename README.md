# Python and High Performance Computing Course

Repository for exercises and coursework from the Python and High Performance Computing course.

## Structure

- `week1/` through `week11/` - Weekly exercises and assignments
- `exercises/` - Additional practice problems
- `HPC_miniproject/` - Separate project repository (not included in this repo)

## Getting Started

Each week's folder contains exercises and solutions related to that week's topics.

## Notes

The `HPC_miniproject/` folder is maintained as a separate GitHub repository and is not included in this repository.
